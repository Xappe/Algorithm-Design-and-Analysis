MB-BRKGA foi uma atividade feita com o objetivo de resolver o problema da Problema da Mochila Binária utilizando a Metaheurística BRKGA (algoritmo genético de chave aleatória tendenciosa).

MB-BRKGA was an activity done with the objective of solving the Binary Knapsack Problem using the BRKGA Metaheuristic (biased random key genetic algorithm).
